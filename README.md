# Customer Segmentation using K-Means Clustering

## Final Year MCA Project
Specialization: Data Science & Artificial Intelligence

---

## Project Overview

Customer segmentation is an essential data science technique used by organizations to group customers based on purchasing behaviour. This project implements an intelligent and interactive customer segmentation system using the K-Means clustering algorithm.

The system allows dynamic dataset upload and automatically performs preprocessing, feature selection, clustering, visualization and customer segment interpretation through a professional Streamlit dashboard.

This project demonstrates the practical implementation of unsupervised machine learning for real-world business analytics applications.

---

## Project Objectives

• Apply unsupervised machine learning (K-Means clustering) for customer segmentation  
• Automatically detect numerical features from uploaded datasets  
• Perform missing value handling and preprocessing  
• Identify optimal number of clusters using Elbow Method  
• Generate customer segment interpretations automatically  
• Build an interactive real-time analytics dashboard using Streamlit  
• Support dynamic dataset upload instead of static dataset dependency  
• Provide business-level insights from clustering results

## Machine Learning Algorithm Used

K-Means Clustering Algorithm

K-Means is an unsupervised machine learning algorithm that groups data points into clusters based on similarity.

In this project it is used to segment customers into:

High Value Customers  
Medium Value Customers  
Low Value Customers  

based on purchasing behaviour patterns.
